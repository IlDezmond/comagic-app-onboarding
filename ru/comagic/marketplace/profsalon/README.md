
### Интеграция с ПрофСалон
<br></br>

 **Ценность**<br></br>   
 
Решение позволяет передавать в наш кабинет заявки, оставленные клиентами в форме Онлайн-записи, а также сделки из ПрофСалона, для дальнейшего построения Сквозной аналитики.
<br></br>

**Какие данные передаются**<br></br>  

Данные получаемые по сделкам:  
- сделки: сумма сделки, название, даты изменения и тд; 
- воронка продаж и ее этапы;
- контакты;
- ответственный менеджер;
- дополнительные поля.  
<br></br>
Данные получаемые по заякам:  

- контактные данные (имя,телефон,email)  
- дату и время создания  
- передача обращений в дефолтную РК или источник <br></br> </br>

### Подключение интеграции с Онлайн-виджетом <br></br> 
 **Необходимые компоненты для работы интеграции**<br></br>  
- Загрузка оффлайн-заявок из внешней системы в нашем сервисе <br></br>
 
<details>
  <summary style="font-weight:bold"> Шаги по подключению </summary> <br></br>  
 
1. Нажмите "Активен" на этой странице.
2. Заполните настройки интеграции  <br></br>

**Авторизация**<br></br>

a. Зайдите в ПрофСалон в раздел Маркетплейс. В нем необходимо найти интеграцию с CoMagic и подключить ее.<br></br> 
После подключения будут выведены данные для авторизации (токен, ID салона, логин салона) <br></br>
b. Необходимо внести эти данные в раздел Авторизация в настройках интеграции.
![image](profsalon_auth.gif) <br></br>

**Настройте Webhook в Профсалон**<br></br>

В Профсалон необходимо настроить Webhook на создание/изменение сделки. <br></br>
Для этого необходимо направить запрос  в ПрофСалон с просьбой создать хук на наш URL (указан в поле Webhook url).
![image](profsalon_hook_url.png)

Если в настройках ПрофСалона есть блок Интеграции → Вебхуки на внешний сервер , то это можно сделать самостоятельно.

![image](profsalon_hook.jpg) <br></br>

**Переключатели:**<br></br> 
a. Нажмите "Подключить интеграцию с Онлайн-виджетом". После этого будут выведены остальные настройки.<br></br>
b. "Рекламная компания/источник" - необходимо выбрать какую сущность использовать для обращений без сессии. По умолчанию выбрана Рекламная компания (переключатель не прожат), при прожатии выбирается Источник.<br></br>  
В зависимости от положения маркера выводится либо список РК из личного кабинета клиента, либо список источников и сайтов. Необходимо указать какую РК/источник и сайт используем в случае отсутствия сессии.<br></br>  

3. Нажмите сохранить. <br></br>

После подключения интеграции заявки будут попадать в  Сырые данные -> Обращения и цели.  
Для проверки корректности работы интеграции оставьте тестовую запись в виджете ПрофСалона.

<br></br>

</details>

  ### Подключение интеграции по передаче сделок<br></br>  

**Необходимые компоненты для работы интеграции** <br></br>  
- Сквозная аналитика в нашем сервисе <br></br>

<details>
  <summary style="font-weight:bold">  Шаги по подключению </summary> <br></br> 

1. Нажмите "Активен" на этой странице.
2. Заполните настройки интеграции <br></br>

**Авторизация** <br></br>
a. Зайдите в ПрофСалон в раздел Маркетплейс. В нем необходимо найти интеграцию с CoMagic и подключить ее.<br></br> 
После подключения будут выведены данные для авторизации (токен, ID салона, логин салона) <br></br>
b. Необходимо внести эти данные в раздел Авторизация в настройках интеграции.
![image](profsalon_auth.gif) <br></br>

**Настройте Webhook в Профсалон** <br></br>

В Профсалон необходимо настроить Webhook на создание/изменение сделки.<br></br> 
Для этого необходимо направить запрос  в ПрофСалон с просьбой создать хук на наш URL (указан в поле Webhook url).
![image](profsalon_hook_url.png)

Если в настройках ПрофСалона есть блок Интеграции → Вебхуки на внешний сервер , то это можно сделать самостоятельно.

![image](profsalon_hook.jpg) <br></br>


**Переключатели:**  
a. Нажмите "Подключить интеграцию по передаче сделок". После этого будут выведены остальные настройки.<br></br>
b. "Дефолтная воронка" - при прожатии все сделки будут передаваться в воронку по умолчанию "Сделки из ПрофСалона". В противном случае будет создана воронка с названием салона, с которым подключена интеграция.<br></br>

3. Нажмите сохранить.<br></br>

После подключения интеграции сделки будут попадать в  Сырые данные -> Сделки.  
Для проверки корректности работы интеграции создайте тестовую сделку(запись) в ПрофСалоне.

</details>
